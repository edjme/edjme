# Установка профильного README

1. На GitHub создайте **публичный** репозиторий с точным именем `edjme`.
   Полный адрес должен стать: `edjme/edjme`.
2. Отметьте **Add a README file**.
3. Загрузите содержимое этого комплекта:
   - `README.md`
   - папку `assets`
4. Убедитесь, что файлы находятся в ветке `main`.
5. Откройте страницу `https://github.com/edjme` — оформление появится автоматически.

Важно: репозиторий должен быть публичным и называться точно так же, как ваш GitHub-логин.
